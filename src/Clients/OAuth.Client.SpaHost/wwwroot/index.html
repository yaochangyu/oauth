PKCE 流程說明

  1. 瀏覽器生成隨機 code_verifier，計算 code_challenge = SHA256(verifier)
  2. 導向 AuthServer /connect/authorize，帶上 code_challenge
  3. 使用者登入後，AuthServer 以 redirect_uri?code=... 回呼
  4. SPA 以 code + code_verifier 向 /connect/token 換取 AT / RT
  5. Token 存於 sessionStorage（不寫入 Cookie）
